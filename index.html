About Me:


     What drew me to computer science is persuit of learning and mastery. I am the type of person that enjoys continuous learning and applying my skillset to solving problems. The ability to work in groups to solve problems always gives me a sense of accomplishment and comradery between myself and colleagues. As of now, I have been mainly working with C, C++, Python, HTML/CSS, JavaScript, Java and SQL for school and side projects. Although, I am determined to learn any technology that will help my side-projects and advance my career goals. My goal after this program is to work as a software engineer to further my skills as a developer.

     I have started my last quarter in my post-bachelor program at Oregon State University and have found a true passion for computer science and programming. I also hold a degree is in Biological Sciences from Arizona State University where I developed molecular and microbiological skills working as a Research Assistant (RA). During my time as a RA, I would often need to problem solve to figure out which biotechnological procedures to use as well as the best method to synthesize proteins. Through this experience, I learned effective communication with my peers, group problem solving, and researching/implementing new methods in my work. Effective skills such as these have been a tremendous help in my computer science and programming studies.

     Some of my recent projects include a Python Wallpaper Scraper: An application that uses the Scrapy python framework to scrape websites and download high-resolution wallpapers and randomly chooses one wallpaper to be set as the desktop background for Windows machines. Chemical Database Project: A fullstack project for class that modeled a chemical inventory database. The front-end was a website interface for the database and was created using HTML/CSS/Javascript and the Bootstrap4 framework. Node.js and MariaDB were used for the back-end. This Personal Website: Built simple website for personal use using HTML/CSS/JavaScript and the Bootstrap4 framework. Used reactive website design principles to make website function for smaller screen sizes (tablet and phone).
